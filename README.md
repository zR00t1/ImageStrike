## 🎈简介

ImageStrike是一款使用PyQT5开发，用于CTF中图片隐写的综合利用工具

<img src="imgs/1.png" style="zoom:60%;" />

## 🎗️环境依赖

```
java
```
[Visual C++ Redistributable Packages for Visual Studio 2013](https://www.microsoft.com/zh-cn/download/confirmation.aspx?id=40784)
